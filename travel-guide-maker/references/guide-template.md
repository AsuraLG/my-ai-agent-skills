# 攻略正文模板（markdown/guide.md）

将此结构作为 `markdown/guide.md` 的起点，根据实际笔记素材填充内容。

```markdown
> 适合人群：xxx。数据来源：笔记整理，整理时间 YYYY-MM-DD。

## 数据来源说明

- 采集时间：
- 关键词：

# 使用说明

- 本攻略基于公开内容整理，请以现场信息为准。
- 景点开放时间、票价等信息可能变化，建议出发前再次确认。
```

**Markdown 语法支持：**
- `#` / `##` / `###` 标题（最多三级）
- `- ` 无序列表
- `1.` 有序列表
- `> ` 引用块（用于高赞评论摘录）
- `---` 分隔线
